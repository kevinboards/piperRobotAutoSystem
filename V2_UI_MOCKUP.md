## Keyboard Shortcuts

### Playback
- **Space** - Play/Pause
- **Esc** - Stop
- **Home** - Go to start
- **End** - Go to end
- **[** - Previous clip
- **]** - Next clip

### Speed
- **Ctrl+1** - 1.0x speed
- **Ctrl+2** - 2.0x speed
- **Ctrl+4** - 4.0x speed

### Timeline Navigation
- **+/-** - Zoom in/out
- **Ctrl+0** - Zoom to fit
- **Left/Right Arrow** - Move playhead
- **Ctrl+Left/Right** - Jump to clip boundaries

### Editing
- **Ctrl+C** - Copy clip
- **Ctrl+V** - Paste clip
- **Ctrl+D** - Duplicate clip
- **Delete** - Remove clip
- **Ctrl+K** - Split at playhead
- **Ctrl+Z** - Undo
- **Ctrl+Y** - Redo

## Example Workflows

### Workflow 1: Simple Sequence
```
1. Record "Pick Part"
2. Record "Move to Station"
3. Record "Place Part"
4. Drag all three to timeline
5. Add 2s gap between each
6. Click Play - robot performs full sequence
```

### Workflow 2: Complex Assembly
```
1. Load existing recordings from library
2. Arrange 10 clips on timeline
3. Trim unwanted portions
4. Speed up travel moves (2x)
5. Slow down precise operations (0.5x)
6. Add gaps for sensor waits
7. Save as "Assembly_Sequence.ppt"
8. Play full timeline (85 seconds)
```

### Workflow 3: Iterative Refinement
```
1. Create initial timeline
2. Play and observe
3. Adjust clip trims
4. Adjust speeds
5. Reposition clips
6. Add/remove gaps
7. Test again
8. Repeat until perfect
9. Export as single .ppr
```
